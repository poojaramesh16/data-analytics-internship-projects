#  Data Analytics Internship Projects

This repository contains four data analytics projects completed during my internship. Each project focuses on solving real-world business problems using data analysis, visualization, and basic machine learning techniques. The goal of these projects is to extract meaningful insights from data and support better decision-making.

---

##  Project 1: HR Analytics – Employee Attrition Analysis

This project focuses on understanding why employees leave an organization. By analyzing employee data such as job role, salary, education, and work conditions, patterns related to attrition were identified.

* Performed data cleaning and preprocessing
* Conducted Exploratory Data Analysis (EDA)
* Visualized attrition trends based on different factors
* Identified key reasons for employee resignation

 **Outcome:**
This analysis helps organizations take preventive measures to improve employee satisfaction and reduce attrition.

---

##  Project 2: E-Commerce Sales and Customer Behavior Analysis

This project analyzes e-commerce transaction data to understand customer purchasing behavior and sales performance.

* Analyzed product-wise and category-wise sales
* Identified top-selling products and revenue drivers
* Studied customer behavior and buying patterns
* Visualized trends using graphs and charts

 **Outcome:**
The project provides insights that help businesses improve sales strategies and understand customer preferences.

---

##  Project 3: Telecom Customer Churn Analysis

This project aims to identify customers who are likely to stop using telecom services (churn).

* Cleaned and processed customer data
* Analyzed churn patterns based on usage and demographics
* Visualized differences between churned and active customers
* Identified important factors affecting customer retention

 **Outcome:**
Helps telecom companies reduce customer loss by targeting high-risk users with better services or offers.

---

##  Project 4: Retail Business Performance and Profitability Analysis

This project focuses on analyzing retail business data to evaluate overall performance and profitability.

* Analyzed sales and profit across different product categories
* Identified high-performing and low-performing products
* Studied sales trends over time
* Visualized data to support business insights

 **Outcome:**
Provides recommendations to improve profit, manage inventory better, and optimize business operations.
